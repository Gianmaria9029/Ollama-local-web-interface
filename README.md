# Ollama Local Web Interface

Interfaccia web leggera per usare modelli locali tramite Ollama.

## Funzioni
- chat locale via API Ollama
- streaming risposta
- selezione modelli
- prompt di sistema modificabile
- esportazione conversazioni
- supporto blocchi <think>

## Requisiti
- Ollama installato e attivo
- almeno un modello disponibile

## Avvio
1. Avvia Ollama
2. Apri `index.html`
3. Inserisci host (default localhost)
4. Seleziona modello
5. Inizia la chat

## Limiti
- nessuna autenticazione
- uso locale/LAN
- no multiutente reale

## Note
Progetto pensato per demo, corsi e test modelli locali.
